# KnowMat

KnowMat 是一个面向材料科学文献的 AI 提取流水线，用于将 PDF、Markdown 与纯文本中的实验与材料信息转换为结构化数据。

文档涵盖项目概览、环境配置、平台支持、贡献指南与变更记录。
